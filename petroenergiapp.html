Aplikasi Pencatatan Keluar Masuk Migas

Aplikasi mobile untuk mencatat dan melacak distribusi migas secara akurat dan efisien.

Overview

Sebuah aplikasi mobile untuk mencatat, memantau, dan melacak distribusi migas secara digital. Dilengkapi fitur laporan keluar masuk migas, dan notifikasi pergerakan migas, membantu mengurangi kesalahan manual dan meningkatkan efisiensi logistik di sektor energi.

Teknologi

  • Mobile: CapacitorJs, VueJs
  • Database: Firebase Realtime DB

Proses Pengerjaan

  1. Pemetaan alur distribusi migas.
  2. Pengembangan modul: input data migas masuk dan keluar, laporan.
  3. Uji coba di lapangan dengan tim logistik.
